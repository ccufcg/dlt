# Desenvolvimento em Blockchain com Hyperledger Besu

Este repositório reúne os exemplos práticos desenvolvidos durante as aulas da disciplina **Desenvolvimento em Blockchain**, com foco na plataforma **Hyperledger Besu**. Os códigos estão organizados em subdiretórios temáticos e podem ser executados diretamente no ambiente de desenvolvimento oficial:


Antes de executar os exemplos, certifique-se de que o ambiente 🔗 [bc101-dev-env](https://github.com/ccufcg/bc101-dev-env) está corretamente configurado. Ele fornece:

- Nós Hyperledger Besu pré-configurados
- RPC HTTP habilitado para testes locais 


## 📁 Conteúdo

| Diretório | Descrição |
|----------|-----------|
| `na10a`  | Exemplos de contratos inteligentes escritos em Solidity |
| `na10b`  | Scripts de interação com a blockchain usando Web3.js (JavaScript) |
| `na10c`  | Scripts de interação com a blockchain usando Web3.py (Python) |
